Ubuntu Quick Setup Scripts

If you are using this script on a Ubuntu machine where you have installed Python directly, without a virtualenv or Anaconda, then you can directly use this script


NOTE: If anything goes wrong, and you change a CMake setting and you'd like to re-make the opencv build, then make sure you do

$ sudo rm -rf opencv/build opencv/CMakeLists.txt
$ cd opencv
$ git reset --hard HEAD
$ cd ..

If even this doesn't work, then completely delete the opencv and opencv_contrib folders and run the shell script again

NOTE2: If you are using Anaconda Python, then, first remove it from your environment variables (as opencv finds an incomplete QT in the anaconda folder), then manually set python 2 and 3 paths in the corresponding fields in ccmake
